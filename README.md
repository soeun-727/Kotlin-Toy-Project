## 기능 소개
- 스플래시 화면이 먼저 나오도록 구성
- 메인 화면은 카테고리를 선택할 수 있는 버튼 12개와 랜덤 식당 추천 버튼으로 구성
- 각 카테고리를 클릭하면 해당 카테고리의 식당이 리스트업 됨 (RecyclerView로 작성)
- 식당 이름 클릭 시 네이버지도 링크로 이동하도록 함
- 랜덤 식당 추천 버튼 클릭 시 팝업으로 식당 하나가 추천됨
- "식당으로 이동" 클릭 시 네이버지도 링크로 이동
